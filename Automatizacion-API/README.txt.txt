# Petstore API Automation Test (OPCIÓN 2 DE APIS)

## Descripción del proyecto

Este proyecto es una prueba técnica para el proceso de selección de QA Automation Engineer en Devsu.
Contiene pruebas automatizadas de API utilizando Cypress para validar la API pública de Petstore.
En las pruebas se verifica la creación de una mascota, consultar esa mascota, actualizar los datos de la mascota 
y la consulta de la mascota por el atributo de estado.

API utilizada:
https://petstore.swagger.io/

Framework de automatización
Cypress

---

## Tecnologías utilizadas

- Node.js
- Cypress
- JavaScript

---

## Instalación del proyecto

1. Clonar el repositorio
git clone https://github.com/juangiraldor/Devsu.git

2. Instalar dependencias
npm install

3. Ejecución de las pruebas

Cypress en modo interactivo: 

npx cypress open

Cypress en modo headless: 

npx cypress run
