# Prueba flujo de compra en saucedemo.com (OPCIÓN 2 AUTOMATIZACIÓN E2E)

## Descripción del proyecto

Este proyecto es una prueba técnica para el proceso de selección de QA Automation Engineer en Devsu.
Contiene pruebas automatizadas E2E utilizando Cypress para validar el flujo de compra dentro de el 
e-commerce de prueba saucedemo.com.
En las pruebas se verifica la autenticación del usuario standard_user, agregar 2 productos al carrito, 
visualizar el carrito, completar el formulario de compra y finalizar la compra hasta su confirmación.
y la consulta de la mascota por el atributo de estado.

Sitio utilizado:
https://www.saucedemo.com/

Framework de automatización
Cypress

---

## Tecnologías utilizadas

- Node.js
- Cypress
- JavaScript

---

## Instalación del proyecto

1. Clonar el repositorio
git clone https://github.com/juangiraldor/Devsu.git

2. Instalar dependencias dentro de la carpeta Automatizacion-E2E
npm install

3. Ejecución de las pruebas

Cypress en modo interactivo: 

npx cypress open

Cypress en modo headless: 

npx cypress run
